Scalable Reconstruction Framework - Not Enough Functions
========================================================

SRF-NEF (NEF in short) is a light weight computing application framework.

.. _Tech-Pi: https://www.tech-pi.com/
  
  
Install
-------

Install and update using `pip _`:

.. code-block:: text

    pip install srfnef
