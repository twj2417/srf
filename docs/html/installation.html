Installation¶

Python Version¶

We recommend using the latest version of Python 3. This package was develped and tested in a Platform with Python 3.6.5.

Dependencies¶

Besides common Python packages, we used these packages below to optimize SEF-NEF.

  • deeddish has a function that converts your Python data type into a native HDF5 hierarchy.

Install SRF-NEF¶

Within the activated environment, use the following command to install SRF-NEF:

pip install -r srfnef

Flask is not installed. Check out the QuickStart or go to the Documentation Overview.
